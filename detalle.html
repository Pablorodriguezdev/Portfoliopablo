Resumen Técnico

Servidor Multimedia Web es una aplicación web diseñada para gestionar y visualizar colecciones de películas de forma privada y personalizada.

🔹 Backend: Spring Boot + Java con base de datos H2 embebida (persistente en disco, directorio /data).

🔹 Frontend: Vue.js + Tailwind CSS para una interfaz moderna, responsiva y adaptable a cualquier dispositivo.

🔹 Funcionalidades: carga de archivos locales, tema claro/oscuro, separación de usuarios, progreso guardado, anotación de películas vistas y reproducción directa en navegador.
